# 02-运算符与表达式

算术运算符： + - \* / %

关系运算符： == != > < >= <=

逻辑运算符： 与`&&` 或`||` 非`!`

位运算符：

| 运算符    | 描述                                                                                   |
| --------- | -------------------------------------------------------------------------------------- |
| `&`       | 参与运算的两数各对应的二进位相与（两位都为1时结果为1，否则为0）                        |
| `\|     ` | 参与运算的两数各对应的二进位相或（两位中至少一个为1时结果为1，否则为0）                |
| `^`       | 参与运算的两数各对应的二进位相异或，当两对应的二进位相异时，结果为1（两位不一样则为1） |
| `<<`      | 左移n位就是乘以2的n次方。`a << b`是把a的各二进位全部左移b位，高位丢弃，低位补0         |
| `>>`      | 右移n位就是除以2的n次方。`a >> b`是把a的各二进位全部右移b位                            |

对于位运算符有点疑惑，还有iota的用法

## iota的用法

`iota`是一个预定义常量，在`const`声明中，每遇到一个新的常量声明行，`iota`的值就会递增（从0开始）

【例子】权限位掩码

- 第一行：writePermission = 1 << iota，此时iota = 0，所以 1 << 0 = 1。因此 writePermission = 1
- 第二行：readPermission 没有显式赋值，会隐式重复上一行的表达式（1 << iota），此时iota已经增加到1，所以 1 << 1 = 2。即 readPermission = 2
- 第三行：executePermission 同理，iota = 2，所以 1 << 2 = 4

```go
const (
	writePermission = 1 << iota  // 0001
	readPermission  // 0010
	executePermission  // 0100
)
```

可以使用下划线`_`跳过某些值

```go
const (
    _  = iota              // 忽略0
    KB = 1 << (10 * iota)  // 1 << 10 = 1024
    MB                     // 1 << 20
    GB                     // 1 << 30
)
```

可以用于定义枚举，`iota`在每次`const`声明块中都会重置为0，下一个块重新计数

```go
type Weekday int
const (
    Sunday Weekday = iota  // 0
    Monday                 // 1
    Tuesday                // 2
    // ...
)
```
